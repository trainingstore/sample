Important HTML tags

This page is designed as a quick reference for some of the common tags covered in Enough HTML. In the process of making it, we'll learn how to make HTML tables via table and related tags.

The tables below don't include all HTML tags, but they do list many of the most important ones.

Document Tags

Tag Name Purpose
html html tells the browser that this is an HTML document
head head includes a title for the document, scripts, styles, meta information, and more.
body body contains all the contents of an HTML document, such as text, hyperlinks, images, tables, lists, etc.
title title defines the title of the document
meta meta Metadata is data (information) about data. It provides metadata about the HTML document. Metadata will not be displayed on the page, but will be machine parsable
style style is used to define style information for an HTML document.

Block Elements

Tag Name Purpose
h1–h6 headings include a heading (levels 1–6)
p paragraph include a paragraph of text
table table include a table
tr table row include a row of data
th table header make a table header
td table data include a table data cell
div division define block-level section in document
header header label the page header
ol ordered list include ordered lists
ul unrdered list include unordered lists
li list include a list item
blockquote block quotation show formatted quotation
br break enter line break

Inline Elements

Tag Name Purpose Example Result
em emphasized make emphasized text <em>technical sophistication</em> technical sophistication
strong strong make strong text <strong>at least a billion people</strong> at least a billion people
a anchor make hyperlink <a href="https://www.tutorialspoint.com/html/html_basic_tags.htm"> HTML tags </a> HTML tags
img img include an image <img src="https://www.tutorialspoint.com/html/images/html-mini-logo.jpg" alt="HTML tags" >
code code display a code <code>var x = 100;</code> var x = 100;
span span define inline section in document <span>book</span>
